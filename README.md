UniTrack – An Academic Progress Tracker

A full-stack web application built using Flask and SQLite that gives
students the ability to manage semesters, courses, tasks, and grades.

## How to Run

1. Install Python 3.x
2. Initialize the database:     python init_db.py
3. Start the application:       python app.py
4. Open a browser and go to:    http://127.0.0.1:5000


Features:
- Semester, course, and task management
- Grade calculations (earned, remaining, course average)
- Task sorting by due date and course

Technologies Used:
- Python (Flask)
- HTML/CSS
- SQLite
